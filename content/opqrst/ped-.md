**来源：**

1. **拉丁语 pes, pedis**
2. **希腊语 paidos**

**含义：**

1. **脚，足** (来自拉丁语)
2. **儿童** (来自希腊语)

---

# pedal (来自拉丁语)

**pedal：踏板；用脚踏**

ped- (拉丁语 pes, pedis)，意为 脚，足

-al (名词/形容词后缀)

#al 

（指用脚操作的装置，或用脚踩踏的动作）

# pedestrian (来自拉丁语)

**pedestrian：行人；平淡无奇的**

ped- (拉丁语 pes, pedis)，意为 脚，足

-estrian (名词/形容词后缀)

#estrian 

（字面意思为“用脚走路的人”，引申为步行者；也可指缺乏想象力，平庸的）

# impediment (来自拉丁语)

**impediment：障碍，阻碍**

im- (前缀，in-的变体)，表示“进入，在……里面”

ped- (拉丁语 pes, pedis)，意为 脚，足

-ment (名词后缀)

#ment 

（字面意思为“在脚下设置障碍”，引申为阻碍前进的事物）

# expedite (来自拉丁语)

**expedite：加速，促进**

ex- (前缀)，表示“出，离开”

#ex 

ped- (拉丁语 pes, pedis)，意为 脚，足

-ite (动词后缀)

#ite 

（字面意思为“使脚离开障碍”，引申为加快进程，促进）

# centipede (来自拉丁语)

**centipede：蜈蚣**

centi- (前缀)，意为 百

#centi 

ped- (拉丁语 pes, pedis)，意为 脚，足

（字面意思为“一百只脚”，指多足的节肢动物）

# pediatrician (来自希腊语)

**pediatrician：儿科医生**

pedi- (希腊语 paidos)，意为 儿童

iatr- (希腊语 iatros)，意为 医生

-ian (名词后缀)

#ian 

（专门治疗儿童疾病的医生）

# pedagogy (来自希腊语)

**pedagogy：教育学，教学法**

ped- (希腊语 paidos)，意为 儿童

agog- (希腊语 agogos)，意为 引导，领导

-y (名词后缀)

#y 

（字面意思为“引导儿童”，指教育的理论和实践）

# encyclopedia (来自希腊语)

**encyclopedia：百科全书**

en- (前缀)，表示“在……内”

#en 

cycl- (希腊语 kyklos)，意为 圆圈，循环

ped- (希腊语 paideia)，意为 教育，知识

-ia (名词后缀)

#ian 

（字面意思为“在循环的教育中”，指涵盖各学科知识的综合性参考书）
