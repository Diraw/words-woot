**来源：** 拉丁语 **struere**

**含义：建造，构成**

# structure

**structure：结构，构造；建造**

-ure (名词后缀)

#ure 

（指事物各部分相互连接和排列的方式；也可指建造某物）

# construct

**construct：建造，构成；构想**

con- (拉丁语 con-)，意为 共同，一起

#con 

（指建造建筑物或构筑物；也可指在头脑中形成想法或理论）

# destruct

**destruct：破坏，毁灭**

de- (拉丁语 de-)，意为 向下，去除

#de 

（指彻底破坏或摧毁某物）

# instruction

**instruction：指示，指令；教学**

in- (拉丁语 in-)，意为 进入，向内

#in

-ion (名词后缀)

#ion 

（指给予的指导或命令；也可指教授知识和技能的过程）

# obstruct

**obstruct：阻碍，妨碍**

ob- (拉丁语 ob-)，意为 反对，对着

#ob

（指阻挡或妨碍某物的通过或进展）
