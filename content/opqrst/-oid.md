**来源：** 希腊语 **-oeidēs** (οειδής)

**含义：具有……的形状/形式/样子**或**像……的**

# android

**android：像人一样的，引申为人形机器人**

[andr-](andr-.md) (希腊语anēr)，意为 人、男性

# ovoid

**ovoid：蛋形的**

[ov-](ov-.md) (拉丁语 ovum)，意为 卵、蛋

# spheroid

**spheroid：类球体，椭球体**

[spher-](spher-.md) (希腊语 sphaira)，意为 球

# tabloid

**tabloid：浓缩的药片，引申为版面较小、内容浓缩或通俗的小报**

tabl- (源自 tablet），意为 药片、小块，引申为 浓缩、紧凑的
# polaroid

**polaroid：偏振片、或品牌名称（宝丽莱拍立得）**

polar- (源自 polarize），意为 使偏振
# hominoid

**hominoid：类人动物、猿(ape)**

homin- (拉丁语 homo)，意为 人
# thyroid

**thyroid：甲状腺（因形状得名）**

[thyr-](thyr-.md) (希腊语 thyreos)，意为 盾牌
# hyoid

**hyoid：舌骨（因形状得名）**

hy- (希腊语字母 upsilon 的形状，大写Υ，小写υ)
# alkaloid

**alkaloid：生物碱**

alkal- (阿拉伯语 al-qaly)，意为 灰烬、植物灰，引申为 碱性的

（因为最初的碱是从植物灰中提取的）
# steroid

**steroid：类固醇**

ster- (希腊语 stereos)，意为 坚固的、立体的、固体的

（指其分子结构是坚固的环状结构）
# opioid

**opioid：阿片类药物（美国的止痛药危机）**

opi- (源自希腊语 opos)，意为 植物汁液，特指 罂粟汁（即鸦片）
# factoid

**factoid：（仅因出现在出版物上而被信以为真的）仿真陈述**
