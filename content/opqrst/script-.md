**来源：** 拉丁语 **scribere**

**含义：写**

# script

**script：剧本，手稿；文字，字体**

script- (拉丁语 scribere)，意为 写

（指戏剧、电影的文本，或手写的稿件，也可指书写系统或字体）

# describe

**describe：描述，描写**

de- (前缀)，表示“向下，完全”

#de 

-scribe (拉丁语 scribere)，意为 写

（字面意思为“写下来”，引申为用文字或语言详细描绘）

# subscribe

**subscribe：订阅，订购；同意，赞成**

sub- (前缀)，表示“在……下面”

#sub 

-scribe (拉丁语 scribere)，意为 写

（字面意思为“在下面写（签名）”，引申为订购服务或同意观点）

# prescribe

**prescribe：开药方；规定，指示**

pre- (前缀)，表示“预先，在……之前”

#pre 

-scribe (拉丁语 scribere)，意为 写

（字面意思为“预先写下”，引申为医生开具药方，或规定规则）

# transcribe

**transcribe：抄写，转录；改编**

trans- (前缀)，表示“穿过，转变”

#trans 

-scribe (拉丁语 scribere)，意为 写

（字面意思为“写过去”，引申为将文字或声音转换成另一种形式）

# postscript

**postscript：附言，（信末的）又及**

post- (前缀)，表示“在……之后”

#post 

-script (拉丁语 scribere)，意为 写

（字面意思为“写在后面”，指信件或文章末尾补充的话）

# manuscript

**manuscript：手稿，原稿**

manu- (源自拉丁语 manus)，意为 手

-script (拉丁语 scribere)，意为 写

（字面意思为“手写的”，指未出版的手稿）

# inscription

**inscription：铭文，碑文；题词**

in- (前缀)，表示“在……里面，在……上”

-script (拉丁语 scribere)，意为 写

-ion (名词后缀)

#ion 

（字面意思为“写在上面”，指刻在石头、金属等上的文字）

# scribble

**scribble：潦草地写，乱涂**

scrib- (拉丁语 scribere)，意为 写

-ble (动词后缀，常表示重复或小动作)

#ble 

（指快速或不认真地书写，导致字迹潦草）
