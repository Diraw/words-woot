**来源：** 拉丁语 **videre**

**含义：看**

# vision

**vision：视力，视觉；远见，幻象**

vis- (拉丁语 videre)，意为 看

-ion (名词后缀)

#ion

（指眼睛看的能力，也可指对未来的预见或脑海中的景象）

# visible

**visible：可见的，明显的**

-ible (形容词后缀)，表示“能够……的”

#ible

（指能够被看到的，或显而易见的）

# visual

**visual：视觉的，视力的**

-ual (形容词后缀)

#ual 

（与视觉或看相关的）

# supervise

**supervise：监督，管理**

super- (前缀)，表示“在……上面”

#super 

（字面意思为“在上面看”，引申为监督和管理）

# revise

**revise：修订，修改；复习**

re- (前缀)，表示“再次，回”

#re 

（字面意思为“再次看”，引申为检查并修改，或为了巩固知识而复习）

# advise

**advise：建议，忠告**

ad- (前缀)，表示“向，去”

#ad 

（字面意思为“去看（并给出意见）”，引申为提供建议）

# video

**video：视频，录像**

-o (名词后缀)

#o 

（指可以观看的影像或录像）

# evident

**evident：明显的，明白的**

e- (前缀，ex-的变体)，表示“出，向外”

#ex 

-vid- (拉丁语 videre)，意为 看

-ent (形容词后缀)

#ent 

（字面意思为“能看出来的”，引申为显而易见的，明确的）

# provide

**provide：提供，供给**

[pro-](pro-.md) (前缀)，表示“向前，预先”

#pro 

-vide (拉丁语 videre)，意为 看

（字面意思为“预先看到并准备”，引申为提供所需物品）
