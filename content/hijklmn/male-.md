**来源：** 拉丁语 **male** (副词，意为 坏地，恶劣地) 或 **malus** (形容词，意为 坏的，恶的)

**含义：坏的，恶的；不好的，错误的**

# malady

**malady：疾病，弊病**

-ady (名词后缀)

#ady 

（指身体或社会上的不适或弊端）

# malevolent

**malevolent：恶意的，恶毒的**

-volent (源自拉丁语 velle)，意为 愿望，意愿

（字面意思为“怀有恶意愿望的”，引申为心怀恶意的）

# malfunction

**malfunction：故障，失灵**

-function (源自拉丁语 functio)，意为 功能，作用

（指功能出现问题，无法正常运作）

# malpractice

**malpractice：渎职，玩忽职守；医疗事故**

-practice (源自拉丁语 practicare)，意为 实践，行为

（指专业人员在工作中行为不当或失职，尤指医生）

# malnutrition

**malnutrition：营养不良**

-nutrition (源自拉丁语 nutrire)，意为 营养

（指身体没有获得足够的或适当的营养）

# malodorous

**malodorous：恶臭的，难闻的**

-odor (源自拉丁语 odor)，意为 气味

-ous (形容词后缀)

#ous 

（指发出难闻气味的）

# malcontent

**malcontent：不满者；不满的**

-content (源自拉丁语 contentus)，意为 满足的

（指对现状不满的人或状态）

# malediction

**malediction：诅咒，咒骂**

[-dict](dict-.md) (源自拉丁语 dicere)，意为 说，讲

-ion (名词后缀)

#ion 

（字面意思为“说坏话”，引申为诅咒或咒骂）
